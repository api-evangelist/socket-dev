Socket Firewall extended protection to editor extensions, blocking malicious installs from VS Code Marketplace and Open VSX before reaching developer machines. Following the compromised Nx Console incident and repeated GlassWorm attacks, the proxy-based enforcement provides an install-time control point for extension marketplaces.
